# Sound Effects

Add the following sound effect files to this directory for interactive hotspots:

- `police-siren.mp3` - Police car siren sound (wee-woo)
- `fire-siren.mp3` - Fire truck siren sound (wailing siren)
- `ambulance-siren.mp3` - Ambulance siren sound (urgent siren)
- `helicopter.mp3` - Helicopter rotor sound (thump-thump)
- `truck-horn.mp3` - Truck horn sound (honk)

## Quick Start

See **`SOUND-EFFECTS-GUIDE.md`** in the project root for detailed instructions!

## Best Sources

### Option 1: Freesound.org (Recommended)
- https://freesound.org/
- Free, high-quality
- Search: "emergency siren", "helicopter rotor", "truck horn"
- Download and rename files

### Option 2: Record Your Own! (Most Fun)
- Use Voice Memos app on iPhone
- Have your son make vehicle sounds
- Convert to MP3
- He'll LOVE hearing himself in the book!

### Option 3: YouTube Audio Library
- https://studio.youtube.com/
- Free, royalty-free sounds

## File Specs

- **Format:** MP3 (web-optimized)
- **Quality:** 128-192 kbps
- **Duration:** 2-5 seconds (short loops)
- **Size:** < 100KB per file

## Once Added

1. Place all 5 MP3 files in this directory
2. Refresh the book page
3. Click on cars in the images
4. Sounds will play automatically!
